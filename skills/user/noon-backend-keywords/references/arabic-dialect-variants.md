## 写法注意事项

1. 字母规范化（后台关键词只用一种写法）：
   - إ / أ / آ → ا (alef)
   - ة → ه (taa marbuta)
   - ى → ي (alef maqsura)

2. 不要混用拉丁 + 阿拉伯（noon 后台字段只能单语种）

3. 避免变音符号（用户基本不输入）

4. 数字写法：
   - 阿拉伯-印度数字 (٤٥) 在 KSA 偶见
   - 拉丁数字 (45) 在 UAE / EG 主流

## 否定词与品牌词（不要进后台）

- 品牌：Apple, Samsung, Nike, Adidas, Loreal, MAC, Dior 等（noon 会因品牌词屏蔽非授权卖家）
- 平台：Amazon, Namshi, noon, Carrefour
- 主观形容词：best, cheap, original, أصلي, أفضل, رخيص
- 时间词：2025, 2026, new, جديد
