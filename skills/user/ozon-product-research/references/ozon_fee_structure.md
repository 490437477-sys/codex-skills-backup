# Ozon Fee Structure (费率结构速查)

> **重要**: Ozon 佣金率、FBO/FBS 物流费、附加费每年都在调整,且按类目差异化。
> **所有数字在使用前必须用 `web_search` 验证最新值**(搜 "Ozon commission rates 2026"、"Ozon FBO fees 2026"、"Ozon FBS tariffs")。
> 本文给出的是典型量级与结构,用于成本建模与可行性判断,不替代 Ozon 后台实时报价。

## 1. Ozon 佣金 (Commission)

按商品类目差异化,典型区间 **5% – 25%**。

| 大类 | 典型佣金率 | 备注 |
|---|---|---|
| Books / 图书 | 5–10% | 最低之一 |
| Fashion (服饰) | 10–15% | 高退货率,实际成本更高 |
| Electronics / 电子 | 8–15% | 大件变体费用另计 |
| Home & Kitchen / 家居 | 10–15% | |
| Beauty / 美妆 | 12–18% | 需 EAC |
| Toys / 玩具 | 10–15% | 需 EAC + 玩具安全认证 |
| Sports / 运动 | 10–15% | |
| Auto parts / 汽车配件 | 10–18% | VIN 匹配要求 |
| Food supplements / 保健品 | 15–20% | 严格监管 |
| Jewelry / 珠宝 | 12–18% | 高价值低退货 |
| Large appliances / 大家电 | 8–12% | 但物流贵 |

> **实操**: 用 `web_search` 拉最新类目表 — Ozon 每年 1–2 次更新。

## 2. FBO 物流费 (Fulfillment by Ozon)

商品从卖家 → Ozon 仓库 → 买家,Ozon 收处理费 + 仓储费 + 配送费。

### 2.1 处理费 (Inbound processing)
按重量 + 尺寸,典型 ₽5 – ₽30/件。

### 2.2 仓储费 (Storage)
- 标准仓储:按立方米/月计费
- 高峰期(Ноябрь–Январь)费率通常翻倍
- 超 60–90 天未售库存:长期仓储附加费

### 2.3 配送费 (Last-mile)
按重量 + 距离,典型 ₽30 – ₽150/件。
- "Доставка завтра" 标签由 FBO 自动获得
- 大件家具/家电:₽300+/件

### 2.4 退货处理费
- 买家退货:FBO 通常免费,但仍消耗仓储 + 处理
- "Выкуп率" (buyout rate) < 70% 会被降低搜索排名

## 3. FBS 物流费 (Fulfillment by Seller)

卖家自行仓储,Ozon 只负责尾程取件 + 派送。

### 3.1 适用场景
- 大件 / 季节性强 / 测试阶段
- 无 FBO 仓储成本压力

### 3.2 典型费用结构
- 拣货 + 打包:卖家自行承担
- Ozon 尾程:按重量区间计费 ₽50 – ₽250
- 退货:卖家承担反向物流 ₽50 – ₽300

### 3.3 FBS 劣势
- 无 "Доставка завтра" 标签
- 搜索排名通常低于 FBO 同款
- 买家等待时间更长 → 转化率打折 20–40%

## 4. 尺寸等级 (Size Tier)

Ozon 按尺寸重量组合划分,影响所有费率:

| Tier | 重量上限 | 尺寸上限 | 适用 |
|---|---|---|---|
| 小型 | < 0.5 kg | 25×20×10 cm | 配饰、电子配件、美妆小件 |
| 标准 | < 5 kg | 50×40×25 cm | 大多数家居、3C、服饰 |
| 大型 | < 25 kg | 100×80×50 cm | 小家电、工具 |
| 超大 | 25+ kg | > 100×80×50 cm | 家具、大家电 — 物流贵 |

> **选品原则**: 优先选择 < 5kg 标准件 — 物流成本可控、买家免运费门槛容易触发(₽3,000+ 商品免运费)。

## 5. 附加费 (Add-on Fees)

### 5.1 强制类目附加费
- 大件商品:+2–5%
- 易碎品:包装审核费
- 奢侈品:鉴定费

### 5.2 推广费 (внутренняя реклама)
- Ozon Performance (类似 Amazon PPC):竞价,典型 ACOS 15–40%
- 站内 Deals / Coupons:卖家自愿承担折扣

### 5.3 Ozon Card 补贴
- 当买家用 Ozon Карта + рассрочка (分期)支付时,卖家可能需补贴 0–2%
- 必须在 listing 中开启该选项才会触发

## 6. Russian VAT 处理

**关键认知**: 自 2023 年起,Ozon 在俄罗斯境内的 B2C 销售中,通常作为 **tax agent** 帮卖家代扣代缴 VAT(НДС)。

- 大多数类目:НДС 20% 由 Ozon 代扣
- 部分类目:跨境商品另有规定
- **影响利润建模**: 卖家标价通常为含税价,VAT 由 Ozon 收走,卖家收到的是税后净额
- **务必用 `web_search` 验证当前 НДС 代扣规则与豁免清单**
