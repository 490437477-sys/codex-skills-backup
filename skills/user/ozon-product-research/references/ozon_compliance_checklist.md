# Ozon 合规与禁限售清单

> **重要**: 俄罗斯进口法规、欧亚经济联盟(EAEU)技术法规、Ozon 平台政策每年都会更新。
> **所有合规要求在上架前必须用 `web_search` 验证最新版本**(搜 "Ozon prohibited categories"、"EAC certification 2026"、"EAEU technical regulation")。

## 1. EAC 认证 (Eurasian Conformity)

欧亚经济联盟(俄罗斯、白俄罗斯、哈萨克、亚美尼亚、吉尔吉斯)强制认证。

### 1.1 强制 EAC 的常见品类

| 品类 | EAC 要求 | 备注 |
|---|---|---|
| 电子产品 | **强制** | TR TS 020/2011(电磁兼容) + TR TS 004/2011(低压) |
| 玩具 | **强制** | TR TS 008/2011 |
| 化妆品 | **强制** | TR TS 009/2011 |
| 食品接触 | **强制** | TR TS 005/2011 |
| 服装(部分) | **强制** | TR TS 007/2011(儿童)+ TR TS 019/2011(成人防护) |
| 个人防护 | **强制** | TR TS 019/2011 |
| 医疗设备 | **强制** | 极严,通常需 Roszdravnadzor 注册 |
| 机械设备 | **强制** | TR TS 010/2011 |
| 汽车配件 | 部分强制 | 需 GOST-R 或 EAC |

### 1.2 EAC 流程

```
产品送检 → 俄罗斯/EAEU 实验室测试 → 颁发 EAC 证书 → 注册到 Ozon listing
```

- 周期:典型 4–8 周
- 费用:单 SKU 约 ₽30,000–₽150,000(品类差异大)
- **建议**:多 SKU / 系列认证,均摊成本

### 1.3 选品启示

- 无 EAC 的品类:书籍、家居非电器、文具、非电子配饰、部分服饰
- 有 EAC 但流程短的:小家电、电子配件(已有同类证书可参考)
- **建议新手从"非 EAC 强制"品类起步**,熟悉后再扩展

## 2. 包装与标签要求

### 2.1 俄语标签(强制)

包装必须包含俄语:
- 商品名称
- 成分 / 材质 / 规格
- 制造商 / 进口商名称 + 地址
- 产地国 (Страна производства, 例: "Страна производства: Китай")
- 生产日期 / 保质期(食品/化妆品)
- 使用说明 / 警告
- 条码 (EAN-13)

### 2.2 商品说明(强制俄语)

- Ozon listing 标题:俄语
- 商品描述:俄语(机器翻译会被降权,**强烈建议母语级**)
- 属性填写:俄语

## 3. Ozon 禁售品类(常见)

**完全禁止**:
- 武器、弹药、爆炸物
- 毒品、精神药品
- 处方药(无 Roszdravnadzor 注册)
- 烟草、电子烟、烟具
- 酒精(需特殊许可)
- 假币、假证件
- 濒危物种制品(象牙、玳瑁等)
- 部分加密货币相关商品

**限制品类**(需特殊许可):
- 保健品 — 需 свидетельство о регистрации
- 婴幼儿食品 — 严格标签 + 配方认证
- 二手商品 — 部分允许,但需明确标注 "б/у"
- 宠物食品 — 部分需兽医许可
- 农药 — 严格管控
- 医疗设备 — Roszdravnadzor 注册

## 4. 制裁 / 合规高风险

虽然俄罗斯进口未完全禁止,但以下情况**强烈不建议**:
- 含美国/欧盟受制裁品牌商标的商品(Apple、Boeing 等品牌名)
- 来自受制裁国家原产地的特定商品
- 双用途商品(military-civilian dual use)
- 加密相关硬件(部分 ASIC miners 受限)

> **风险**: 资金冻结、海关扣押、Ozon 下架、卖家账户封禁。
> **务必用 `web_search` 核实当前制裁清单** + 咨询合规律师。

## 5. 知识产权(IP)

- **不允许** 销售带未授权商标的商品(Nike 假货、Disney 周边等)
- Ozon 对侵权零容忍,直接下架 + 封号 + 扣保证金
- 即使原产国允许,在俄罗斯可能仍构成侵权
- **建议**: 自有品牌 / 公版设计 / 已获授权 IP

## 6. 海关与进口

### 6.1 个人包裹 vs 商业进口

- 单票 < €200 / 月 < 31 kg:通常简化清关(但 2026 规则可能变化)
- 商业进口:需正式报关,缴关税 + НДС
- **Ozon Global 卖家**:Ozon 通常作为进口主体处理

### 6.2 关税 HS Code

每个品类都有 HS Code,决定关税税率(0% – 20%+)。
> 务必用 `web_search` 查目标品类的 HS Code 与关税。

### 6.3 跨境物流建议

- 头程:海运拼柜(慢但便宜) vs 空运(快但贵)
- 末端:Ozon FBO 仓入库(莫斯科、新西伯利亚等)
- 旺季(11–12月)头程务必提前 6–8 周

## 7. 卖家合规自查清单

上架前必查:
- [ ] 商品是否在 Ozon 禁售清单
- [ ] 是否需要 EAC 认证
- [ ] 包装是否含完整俄语标签
- [ ] 商品描述是否为俄语(母语级)
- [ ] HS Code + 关税已确认
- [ ] 无未授权商标 / IP
- [ ] 卖家主体 + 银行账户可正常收 RUB
- [ ] 已确认 НДС 处理方式(Ozon 代扣 / 自行申报)
