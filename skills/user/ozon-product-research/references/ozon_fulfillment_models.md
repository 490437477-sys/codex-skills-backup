# Ozon FBO vs FBS 履约模式对比

## 一句话总结

- **FBO** = 把货提前发到 Ozon 仓库,Ozon 负责仓储 + 拣货 + 配送 — 转化高、但有入库资金占用
- **FBS** = 卖家自己仓,Ozon 派快递上门取件 — 灵活、但没有 "Доставка завтра" 标签

## 详细对比

| 维度 | FBO (Fulfillment by Ozon) | FBS (Fulfillment by Seller) |
|---|---|---|
| 仓储位置 | Ozon 仓库(莫斯科/新西伯利亚等多仓) | 卖家自有仓库 |
| 拣货 + 打包 | Ozon 负责 | 卖家负责 |
| 配送 | Ozon 配送网络 | Ozon 派快递取件后配送 |
| "Доставка завтра" 标签 | **自动获得** | 无 |
| 搜索排名 | 通常较高 | 通常较低 |
| 转化率(同款对比) | 基准 | 通常 -20–40% |
| 入库物流成本 | 卖家承担(海运/空运到 Ozon 仓) | 无 |
| 仓储费 | 按 m³/月 | 无 |
| 现金占用 | 高(需提前备货) | 低(按单发货) |
| 滞销风险 | **高**(长期仓储附加费) | 低 |
| 大件/超重商品 | 部分拒收或加费 | 灵活 |
| 退货处理 | Ozon 处理 + 上架 | 卖家自己处理 |
| 适用阶段 | 稳定出单后主力 SKU | 测款、季节品、大件 |

## 选品与履约模式决策树

```
新品 / 未知需求
   │
   ├── 是大件/超重/易碎? ── Yes ──► FBS (起手)
   │                       │
   │                       └── No
   │
   ├── 是否季节性强(只在 НГ/лето)? ── Yes ──► FBS (避免滞销)
   │
   ├── 单价 < ₽500? ── Yes ──► FBS (FBO 入库成本占比太高)
   │
   └── 标准件 + 全年稳定需求 ──► FBO (主力)
```

## 切换时机

从 FBS → FBO:
- SKU 连续 30 天日均 ≥ 10 单
- 利润率能覆盖 FBO 入库物流 + 仓储
- 商品尺寸重量在 Ozon FBO 接受范围内
- 已通过 EAC/合规审核

从 FBO → FBS:
- 商品销售下滑,库存周转 > 60 天
- 新一代 SKU 替代旧款,清理库存
- FBO 仓储费已超过商品价值

## 跨境测试(无俄罗斯库存)

无 Ozon Global 入驻时,可选 **Cross-border from China**:
- 买家下单 → Ozon 从中国卖家仓库取件 → 跨境到买家
- 配送时间长(15–30 天)
- 转化率低、纠纷率高
- **仅适合测试阶段 + 低 MOQ SKU**

正式做 Ozon.ru,建议 **第一波测款用 FBS → 主力 SKU 切 FBO**。

## Ozon Global(卖家入驻模式)

- Ozon 邀请/允许中国卖家入驻
- 通常以 **跨境 FBO**(货发到 Ozon 中国仓)或 **跨境 FBS** 起步
- 进入 Ozon.ru 主站需通过 Global Selling 通道
- **务必核实当前 Ozon Global 入驻门槛、保证金、回款方式**

## 操作 Checklist

FBO 上架前:
- [ ] 商品尺寸重量 ≤ Ozon FBO 上限
- [ ] 已取得 EAC/所需认证
- [ ] 包装含俄语标签(название, состав, страна)
- [ ] 库存已到 Ozon 仓
- [ ] Russian listing 已发布(标题/描述/图片本地化)
- [ ] 设置目标价格(已扣除 Ozon 佣金 + 物流)

FBS 上架前:
- [ ] 卖家仓库地址已登记
- [ ] 处理时效承诺(通常 1–3 天)
- [ ] 包装标准符合 Ozon 要求
- [ ] Russian listing 已发布
- [ ] 退货流程已规划(退货地址 + 处理 SOP)

## 性能指标

- **Выкуп率 (Buyout Rate)**: 买家实际签收率,目标 > 90%
- **Отмены (Cancellation Rate)**: 卖家主动取消率,目标 < 2%
- **Время обработки (Processing Time)**: 接单 → 交付物流,目标 < 24h
- **Отзывы рейтинг (Seller Rating)**: 目标 > 4.5 / 5.0

> **红线下**: 卖家评分持续 < 4.0 → Ozon 会降低搜索权重甚至下架 listing。
