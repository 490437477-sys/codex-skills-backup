# noon Advertising Strategy

Design a full-funnel noon advertising strategy tuned to MENA seasonality, Arabic keyword behavior, and the three-marketplace footprint.

## Capabilities

- **Channel mix planning**: Sponsored Products (always), Sponsored Brands (where enabled), display retargeting
- **Budget allocation** across noon.sa / noon.ae / noon.com and across awareness / consideration / conversion stages
- **Keyword strategy**: Arabic MSA + dialect coverage (KSA / UAE / EG), exact / phrase / broad structure
- **Bid calibration**: competitive SAR / AED / EGP bids per match type and lifecycle stage
- **ACOS targets** by product stage (launch: 60-80%, mature: 25-35%, sunset: < 20%)
- **Seasonal overlays**: Ramadan (huge), White Friday, back-to-school, Eid, National Day
- **Competitor defensive bidding**: brand-protection campaigns on top competitor names
- **Cross-border leak detection**: identify when noon ads drive traffic that converts on Amazon.ae / Amazon.sa
- **Performance forecasting**: expected clicks, orders, ROAS at given budget levels

## Workflow

### 1. Confirm marketplaces and budget
Which noon markets (sa / ae / com)? Total monthly budget in SAR / AED / EGP (or USD)? Currency split?

### 2. Stage the product portfolio
Bucket every SKU into:
- **Launch** (< 30 days): need discovery, accept 60-80% ACOS
- **Growth** (30-90 days, climbing BSR): scale winners, target 35-50% ACOS
- **Mature** (> 90 days, stable): defend, target 25-35% ACOS
- **Sunset**: harvest, target < 20% ACOS

### 3. Build the campaign structure
For each SKU stage:

| Stage | Campaigns | Goal |
|-------|-----------|------|
| Launch | SP exact (top 20 Arabic + EN keywords) + SP auto + SP brand-defense | Discovery |
| Growth | SP exact + SP product-targeting (top competitor ASINs) + SB category | Scale |
| Mature | SP exact (defend) + SB brand + display retargeting | Defense + LTV |
| Sunset | SP brand only | Margin preservation |

### 4. Allocate budget

Default split for a balanced portfolio:
- Sponsored Products: 70%
- Sponsored Brands: 20%
- Display / off-platform: 10%

Adjust seasonally: bump Sponsored Brands during Ramadan / White Friday (high purchase-intent traffic).

### 5. Set bids
- noon.sa bids typically 20-40% higher than noon.ae (larger market, more competition)
- Egypt bids 30-50% lower than KSA (lower AOV)
- Ramadan: bid up 30% across the board 2 weeks before Ramadan starts

### 6. Plan seasonal overlays
For each calendar peak:

| Event | Window | Action |
|-------|--------|--------|
| Ramadan | 30 days before Eid | +50% budget, focus on gifting / food / home |
| White Friday | last week Nov - early Dec | full-funnel push, biggest deals |
| Back-to-school | mid Aug - mid Sep | stationary, electronics, kids |
| National Day KSA | Sep 23 | patriotic / Saudi-flag themed promos |
| Back-to-school UAE | early Sep | school supplies, electronics |

### 7. Forecast
Expected clicks = budget / avg CPC. Expected orders = clicks x CVR. Expected ROAS = revenue / spend.

## Output

- **Channel mix** with budget split (% and SAR/AED/EGP)
- **Campaign structure** (campaign name, type, budget, bid, target ACOS)
- **Seasonal calendar** with budget multipliers per event
- **Forecast** at conservative / base / aggressive scenarios
