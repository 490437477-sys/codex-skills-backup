## 斋月节奏细分（noon 最重要的事件）

斋月前 60 天 = 准备期。提前采购、压货、铺关键词。

| 阶段 | 时间窗口 | 行动 |
|------|----------|------|
| T-60 | 斋月开始前 2 个月 | 锁定爆款 SKU / 备货 FBN / 拍开箱视频 |
| T-30 | 斋月开始前 1 个月 | 上线 Sponsored Products / 测词 / 优化 Listing |
| T-7 | 斋月开始前 1 周 | 提价 5-10% / 全面投放 / 备货 FBN 100% |
| 斋月 R1-R10 | 斋月头 10 天 | 高峰期 / 投放加码 50% / 监测 ACoS |
| 斋月 R11-R20 | 斋月中段 | 维持高预算 / 测试新关键词 / Sponsored Brands |
| 斋月 R21-R30 | 斋月末段 + 开斋节前 | 礼品 / 服饰 / 儿童 / 礼盒 |
| 开斋节 | 3-5 天 | 礼品需求爆发 / 提早投放 |
| 节后 R+7 | 开斋节后 1 周 | 降价清仓 / 保留品牌词投放 |

## White Friday 节奏（noon 最大促销）

White Friday = noon 的双 11。通常在 11 月最后一周到 12 月第一周。

| 阶段 | 时间 | 行动 |
|------|------|------|
| T-45 | 11 月初 | 锁定主推 SKU / 锁定广告预算 |
| T-14 | 11 月中 | 预热期：低预算高覆盖 / Sponsored Brands 投品类词 |
| T-3 | 11 月倒数第 4 天 | 主会场商品审核通过 / 折扣生效 |
| D-Day | 11 月最后一周 | 全面投放 / 预算 +100% / 全时段投放 |
| 后 3 天 | 12 月初 | 收尾 / 复盘 / 清理库存 |

## 每日投放时段

MENA 用户作息：
- 早高峰：8:00-10:00 (KSA / AE 工作开始)
- 午高峰：12:00-14:00 (午休购物)
- 晚高峰：20:00-24:00 (核心购物时段，KSA / AE 最高)
- 深夜：0:00-2:00 (年轻男性为主，EG 较活跃)

默认开启晚高峰预算 +100% 加码。

## 复盘节奏

每月 1 号：拉月度数据
- ACOS 是否在目标区间
- 关键词覆盖度
- 广告 SKU 占比（建议 70-80%）
- 退货率
- COD 跳单率

每季度：调整策略
- 重新分配 SP/SB/Display 预算
- 调整目标市场权重
- 季节性 SKU 上下架

每半年：策略大复盘
- 整体 ROAS 趋势
- 利润率 vs ACOS 关系
- 竞争对手动作
