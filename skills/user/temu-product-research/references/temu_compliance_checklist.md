# TEMU 合规与禁限售自查清单

> 在确认选品前逐项过一遍。任何一项不通过都需 **先解决再上架**。TEMU 对违规扣款极重（差评 + 滞销 + 强制下架 + 罚款）。

## 一、TEMU 平台层面

- [ ] 该类目在目标站点未被列入 **限售/禁售目录**
- [ ] 不涉及品牌仿货、Logo 抄袭、卡通/明星 IP（迪士尼、Sanrio、奥特曼、NBA/NFL、Nike、Adidas 等）
- [ ] 不属于二手 / 翻新 / 召回产品
- [ ] 不含违禁成分（药品、激素、麻醉、易燃易爆、放射性）

## 二、目标国法律法规

### 美国 (US)

- [ ] 含电子元件 → FCC 认证
- [ ] 儿童用品 (12 岁以下) → CPC + CPSIA + 8 项重金属测试
- [ ] 玩具 → ASTM F963
- [ ] 接触食品 → FDA
- [ ] 含锂电池 → UN38.3 + MSDS
- [ ] 加州市场 → Prop 65 警示标
- [ ] 纺织品 → 纤维成分标 + 洗涤标 + 原产国

### 欧盟 (DE/FR/IT/ES/NL/...)

- [ ] CE 标识（电子、玩具、PPE 等）
- [ ] EPR 注册（包装、电子电器、电池、纺织品分别需要）
- [ ] GPSR (2024 年起强制) 责任人 + 安全文档
- [ ] REACH / RoHS 化学物质合规
- [ ] 玩具 → EN71
- [ ] 纺织品 → 标签 (材质 + 原产国)

### 英国 (UK)

- [ ] UKCA 标识（替代 CE）
- [ ] 玩具 → UKCA + 安全测试
- [ ] 纺织品 → 标签

### 日本 (JP)

- [ ] 含电子元件 → PSE / 电波法 (技适)
- [ ] 食品接触 → 食品卫生法
- [ ] 婴幼儿 → ST 标志 (玩具安全协会)

### 韩国 (KR)

- [ ] KC 认证（电子、儿童用品）
- [ ] 化妆品 → MFDS 申报

### 拉美 (MX/BR)

- [ ] 巴西 INMETRO（电子）/ ANVISA（化妆品、母婴）
- [ ] 墨西哥 NOM 标识
- [ ] 西语/葡语标签

### 中东 (SA/AE)

- [ ] SASO / G-Mark (沙特/海湾电子)
- [ ] 阿语标签
- [ ] 文化禁忌排查（宗教、酒精、猪、人物形象）

## 三、物流端

- [ ] 单件重量在平台允许范围内（全托管对超大超重通常拒收）
- [ ] 不含液体 / 含磁铁 / 含锂电池大件（除非走半托管 + 海外仓）
- [ ] 包装能承受跨境长距离运输（跌落、挤压、温湿度）
- [ ] 包装上印的图案/文字无 IP 风险

## 四、品控与售后

- [ ] 工厂可提供：成分/材质 SDS、检测报告、生产日期、批号
- [ ] 退货率预估（参考同类目）：< 15% / 15–25% / > 25%
- [ ] 售后预案：替换件、安装说明、多语言客服话术

## 五、IP 自查（最常踩坑）

逐项 web_search 验证：

1. 产品图样是否与已注册商标 / 外观专利相似？
2. 标题/描述里是否使用了"Disney style"、"Apple style"等暗示性词汇？
3. 包装上的卡通形象、人物剪影是否商用授权？
4. 字体是否商用授权（Helvetica、思源、商业字库）？
5. 在 USPTO / EUIPO / TMview 上搜过名称是否撞商标？
