# TEMU Profit Analyzer

针对 TEMU 平台特有的「平台核价 + 高退货率 + 质保扣款 + 卖家被动定价」四大利润黑洞，做全托管 / 半托管 / 本地对本地三套利润模型穿透测算。

> **重要前提**：TEMU 的佣金、扣点、核价规则每年都在变。**所有费率、核价基线、质保比例在动笔前必须用 web_search 验证最新值**，不要直接相信历史示例数字。

## Capabilities

- 三种履约模式利润模型并列对比（供货价、平台佣金、头程、尾程、退货、质保、广告）
- TEMU 持续核价压力模拟：首月 → 3 个月 → 6 个月 → 12 个月的售价/利润曲线
- 多币种结算折算与汇率波动缓冲（默认按 USD 统一口径，可切 EUR/GBP/JPY/KRW/MXN/BRL）
- 退货率敏感度分析：10% / 20% / 30% / 40% 退货率对净利的吞噬比例
- 回本周期测算、最低安全库存、单 SKU 盈亏平衡日销
- 「值不值得做」最终结论（绿灯 / 黄灯 / 红灯 判断）

## When to Use

- 用户问「这个 SKU 在 TEMU 上能赚多少」「TEMU 还赚钱吗」
- 用户对比 TEMU 全托管 vs 半托管利润差距
- 用户被 TEMU 核价压价后想判断还值不值得继续卖
- 用户准备上新品需要做利润测算与定价底线
- 用户问 TEMU 退货率多高、扣多少货款、回本要多久
- 用户问 TEMU 净利润率、TEMU 卖家收益、TEMU 货款结算

## TEMU 利润的 4 大隐形吞噬点

1. **核价压力**：同款 / 类似款在 TEMU 上 Top 5 最低价的平均值，平台会持续要求商家降价。**首次核价通常要求降 5% - 15%，3 个月内可能再来一次**。
2. **退货率**：TEMU 退货政策宽松（90 天无理由、部分品类免费退货），鼓励消费者退货，导致商家货损率远高于 Amazon。
3. **质保扣款**：差评 / 破损 / 漏发 / 发错货都会被 TEMU 直接扣货款，且申诉成功率低。
4. **质保金占用**：TEMU 通常冻结 1 - 3 个月质保金，新卖家首批货款回款慢。

## Workflow

### Step 1 - 锁定模式与站點
向用户确认：全托管 / 半托管 / 本地对本地，目标市场（US / EU / JP / KR / MX / BR 等），结算币种，结算账期。

### Step 2 - 采集 6 个输入
1. 拟供货价（CNY 或 USD）
2. 平台拟零售价（USD 或当地币）
3. 产品裸重 + 包装后体积重
4. 头程方式（海运拼柜 / 海运整柜 / 空派 / 快递）
5. 历史或预期退货率
6. 广告预算占营业额比例

### Step 3 - 拆解成本瀑布
按顺序扣除：货品成本 → 头程 → 包装 → 平台佣金 → 质保金 → 尾程（半托管）→ 退货摊销 → 广告 → 汇率损耗。每一步都标百分比和绝对值。

### Step 4 - 核价压力测试
模拟 4 个时间点：上架价 → 平台首次核价后 → 3 个月 → 6 个月。价格被压 5% / 10% / 15% / 20% 时的净利润变化，给出「还能撑多久不亏」的临界月数。

### Step 5 - 退货敏感度
在当前定价下，分别计算退货率 10% / 20% / 30% / 40% 时的净利率，找到盈亏平衡退货率（break-even return rate）。

### Step 6 - 现金流与回本
根据结算账期 + 广告预付 + 头程压款，估算每月净现金流与累计回本天数。

### Step 7 - 给出报价底线与止损线
输出：最低供货价红线、单 SKU 最低日销、现金流回正天数、是否值得做（绿 / 黄 / 红 灯判断）。

## Output

输出主表：`模式 | 供货价 | 售价 | 佣金 | 头程 | 尾程 | 退货摊销 | 质保 | 广告 | 净利 | 净利率`

附表：
- 核价压力曲线（4 个时间点的净利率变化）
- 退货敏感度（4 个退货率档位的净利率）
- 回本日历（第几天累计现金流为正）

最终结论：绿灯（净利率 ≥ 15%）/ 黄灯（5% - 15%）/ 红灯（≤ 5% 或亏损）。

## Quick Mode

只输入：供货价 + 售价 + 重量 + 履约模式 + 目标市场。5 个数字 → 直接输出净利率与「值不值得做」的判断。

## 常见误区

- **只看售价不看核价**：TEMU 上架价 1 个月后大概率会被压，下单前必须用 Step 4 模拟。
- **忽略退货摊销**：TEMU 退货率 20% 是常态，不是异常值。
- **忽略质保扣款**：1% - 5% 的扣款看似小，但叠加退货与核价后可能让净利率从 15% 跌到 5%。
- **把广告当投资**：TEMU 自然流量分配不公平，新品基本要靠广告起量，广告预算要算进成本瀑布而不是「额外支出」。


## 实战 Tips

- **永远跑一次 Quick Mode 再做详细测算**：5 个数字 10 秒出结果，先判断大方向值得做再深入。
- **TEMU 数字每月变**：所有费率 / 扣点 / 核价基线 / 退货率每月都可能微调，建议每 30 天复核一次。
- **多市场分散风险**：单一市场被核价压垮时可以切换到另一市场，履约链路要支持多目的地。
- **保留 10% - 15% 利润缓冲**：所有测算都假设最好情况，实际留出 10% - 15% 的缓冲空间才稳。
