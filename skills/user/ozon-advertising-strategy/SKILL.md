# Ozon Advertising Strategy

Design a full-funnel Ozon advertising strategy tuned to CIS seasonality, Russian keyword behavior, and the three-marketplace footprint.

## Capabilities

- **Channel mix planning**: Ozon Performance (search + product promotion), banner ads, brand shelf
- **Budget allocation** across Ozon.ru / Ozon.kz / Ozon.by and across awareness / consideration / conversion stages
- **Keyword strategy**: Russian MSA + colloquial + transliterated forms, exact / phrase / broad structure
- **Bid calibration**: competitive RUB / KZT / BYN bids per match type and lifecycle stage
- **ACOS targets** by product stage (launch: 50-70%, mature: 20-30%, sunset: < 15%)
- **Seasonal overlays**: New Year (huge), 3.8 Womens Day, 9.1 Knowledge Day, Black Friday
- **Competitor defensive bidding**: brand-protection campaigns on top competitor names
- **Cross-platform leak detection**: identify when Ozon ads drive traffic that converts on Wildberries / Yandex Market
- **Performance forecasting**: expected clicks, orders, ROAS at given budget levels

## Workflow

### 1. Confirm marketplaces and budget
Which Ozon markets (ru / kz / by)? Total monthly budget in RUB / KZT / BYN (or USD)? Currency split?

### 2. Stage the product portfolio
Bucket every SKU into:
- **Launch** (< 30 days): need discovery, accept 50-70% ACoS
- **Growth** (30-90 days, climbing rank): scale winners, target 25-40% ACoS
- **Mature** (> 90 days, stable): defend, target 20-30% ACoS
- **Sunset**: harvest, target < 15% ACoS

### 3. Build the campaign structure
For each SKU stage:

| Stage | Campaigns | Goal |
|-------|-----------|------|
| Launch | Performance search exact (top 20 RU keywords) + Performance auto + brand-defense | Discovery |
| Growth | Performance search exact + Performance product-targeting (top competitors) + Banner | Scale |
| Mature | Performance search exact (defend) + brand shelf + banner retargeting | Defense + LTV |
| Sunset | Performance brand only | Margin preservation |

### 4. Allocate budget

Default split for a balanced portfolio:
- Performance search: 60%
- Performance product promotion: 25%
- Banner / brand: 15%

Adjust seasonally: bump product promotion during New Year / 3.8 / 9.1 (high purchase-intent traffic).

### 5. Set bids
- Ozon.ru bids typically 2-3x higher than Ozon.kz / Ozon.by (larger market, more competition)
- New Year: bid up 30% across the board 2 weeks before New Year starts (Dec 25)
- 3.8 Womens Day: bid up 50% in flowers, jewelry, cosmetics 1 week before Mar 8

### 6. Plan seasonal overlays
For each calendar peak:

| Event | Window | Action |
|-------|--------|--------|
| New Year | Dec 25 - Jan 8 | +80% budget, focus on gifts, electronics, home |
| 3.8 Womens Day | 2 weeks before Mar 8 | +50% budget, flowers, jewelry, cosmetics |
| 9.1 Knowledge Day | 2 weeks before Sep 1 | +40% budget, school, kids, electronics |
| 5.9 Victory Day | 1 week before May 9 | +20% budget, patriotic, gifts |
| Black Friday | Last week Nov | full-funnel push, biggest deals of year |
| 11.11 Singles Day | Nov 11 | emerging but growing |

### 7. Forecast
Expected clicks = budget / avg CPC. Expected orders = clicks x CVR. Expected ROAS = revenue / spend.

## Output

- **Channel mix** with budget split (% and RUB)
- **Campaign structure** (campaign name, type, budget, bid, target ACoS)
- **Seasonal calendar** with budget multipliers per event
- **Forecast** at conservative / base / aggressive scenarios
