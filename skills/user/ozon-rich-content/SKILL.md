# Ozon Rich Content

Build compelling Russian-language Rich Content for the Ozon marketplace. Ozon Rich Content is the closest analogue to Amazon A+ Content in CIS: brand-registered sellers can use modules to tell a richer product story.

## Capabilities

- **Brand program check**: confirm the seller is enrolled in Ozon brand program before recommending Rich Content modules
- **Module selection**: choose from Ozon Rich Content module types (hero, comparison, lifestyle, feature grid, brand story)
- **Russian copy**: write idiomatic Russian with appropriate tone; Russian shoppers value detail and specificity
- **Comparison chart planning**: build feature-vs-feature matrices that resonate with CIS shoppers (e.g. voltage 220V, cold-weather rated, Cyrillic interface)
- **Image briefs**: detailed art direction briefs for designers (composition, color palette, typography, model demographics)
- **Cultural sensitivity**: avoid imagery that conflicts with CIS norms; emphasize family, practicality, durability
- **Mobile-first design**: 70%+ of Ozon traffic is mobile; modules must render well on small screens

## Workflow

### 1. Confirm brand program
Rich Content requires enrollment in Ozon brand program. If the seller is not enrolled, recommend enrollment first.

### 2. Pick the module mix
Default 5-module layout:

| # | Module | Purpose |
|---|--------|---------|
| 1 | Hero banner | Brand promise + lifestyle |
| 2 | Feature grid (4-tile) | Key product features with icons |
| 3 | Comparison chart | You vs competitors / old vs new |
| 4 | Lifestyle image | Product in use (CIS setting) |
| 5 | Brand story | Heritage, mission, customer trust |

Add more modules only if conversion data supports it.

### 3. Write Russian copy
- Russian (primary)
- Russian + English for cross-border
- Keep bullet length <= 80 characters (mobile truncation)
- Use strong action verbs, practical tone
- Russian shoppers value: specs, durability, warranty, customer service availability

### 4. Build image briefs
For each module:
- Dimensions (e.g. hero 1920x720)
- Subject matter
- Color palette (align with brand; popular CIS palettes: red/blue for trust, white/green for clean tech)
- Model demographics if applicable
- Text overlay space (Russian is longer than English; budget accordingly)

### 5. Cultural review
Avoid:
- Western holiday imagery out of season
- Political / military imagery
- Religious imagery (Russia is multi-religious)
- Country flags except for marketing the origin

Prefer:
- Family settings
- Home / hospitality / cooking contexts
- Practical / durable feel
- Cyrillic typography samples

### 6. Mobile QA
Test each module at 360px width. Text must remain readable; imagery must not crop awkwardly.

## Output

- **5-module layout plan** with Russian copy per module
- **Image briefs** (one per module, designer-ready)
- **Cultural compliance checklist** (passed / flagged / action required)
- **Mobile QA notes**

## Quick Mode

If user wants only one module (e.g. comparison chart): return just that module's copy + image brief.
