## 4. Hijri-to-Gregorian Approximation (Ramadan planning)

Use this when the user asks about Ramadan keyword lift but does not specify a year. Lunar calendar drifts ~10-11 days earlier each Gregorian year.

| Hijri Year | Approx. Ramadan start (Gregorian) |
|------------|------------------------------------|
| 1446 AH | ~Feb 28, 2025 |
| 1447 AH | ~Feb 17, 2026 |
| 1448 AH | ~Feb 6, 2027 |
| 1449 AH | ~Jan 26, 2028 |

Always double-check with an authoritative source before locking inventory or PPC dates.

## 5. KSA / UAE / Egypt Spelling Variants

For some categories, the same product has different spellings:

| Concept | KSA / MSA | Egypt | UAE |
|---------|-----------|-------|-----|
| Sneakers | حذاء رياضي | كوتشي | سنيكرز |
| Fridge | ثلاجة | تلاجة / ديب فريزر | ثلاجة |
| Stove | فرن / موقد | بوتاجاز | فرن |
| Cart | عربة | كارت | عربة |
| Awesome / excellent | ممتاز | تحفة | خرافي |

When targeting noon.eg separately, lean into Egyptian variants. For noon.sa + noon.ae, MSA + Gulf dialect is the safer choice.

## 6. Religious / Cultural Sensitivity Keywords

Avoid putting these into Arabic keyword lists unless the product genuinely targets them:

- halal, حلال — only if product is genuinely halal-certified
- muslim, مسلم — avoid as a generic modifier
- modest fashion, عباية — appropriate only for that category
- prayer, صلاة / مصلى — only for prayer-related products
- quran, قرآن — only for Quran-related products

## 7. Bilingual Title Pattern (recommended for noon)

```
[Primary Arabic keyword] | [English brand/model in parentheses]
```

Example:
```
سماعات لاسلكية AirPods Pro 2 مع علبة شحن MagSafe
```

The Arabic keyword leads (matches Arabic search), the English brand term captures branded search. noon indexes both fields.
