# TEMU Rich Content 馃徑

围绕 TEMU 商品详情页规范，设计能提升转化率、降低退货率、适配多语言市场的主图、视频与详情模块。

## Capabilities

- **主图 5 张规划**：白底主图 + 卖点图 + 场景图 + 尺寸图 + 包装图的标准组合
- **视频脚本**：15-30 秒产品视频前 3 秒 hook 与 CTA 节奏
- **详情页模块排版**：场景 / 核心卖点 / 参数表 / 使用步骤 / 资质证书 / 售后保障 6 大模块
- **多语言适配**：英 / 德 / 法 / 西 / 日 / 韩 / 葡 图文本地化
- **退货率导向设计**：用详情页前置预期减少「与想象不符」类退货
- **节庆模板库**：黑五 / 圣诞 / 母亲节 / 复活节主题素材替换

## When to Use

- 用户新链接上架，需要规划详情页全套素材
- 用户详情页跳出率高，需要重做主图与详情模块
- 用户退货率高于类目中位数，需要前置预期型详情
- 用户做节庆活动，需要快速生成活动期素材包

## TEMU 主图与视频规格（关键约束）

### 图片规格

| 位置 | 推荐尺寸 | 比例 | 大小上限 | 备注 |
|---|---|---|---|---|
| 主图（白底） | 800x800 至 1200x1200 | 1:1 | 2 MB | 纯白背景占 ≥ 85% |
| 辅图 2-5 | 800x800 至 1200x1200 | 1:1 | 2 MB | 卖点 / 场景 / 尺寸 / 包装 |
| 详情长图 | 750 宽，高度不限制 | 3:4 / 竖向 | 单图 ≤ 5 MB | 整页总大小 ≤ 20 MB |

### 视频规格

- 时长：15-30 秒为佳，不超过 60 秒
- 比例：1:1 或 3:4，竖屏 9:16 在某些站点支持
- 分辨率：720p 起步，推荐 1080p
- 格式：MP4（H.264），大小 ≤ 50 MB

> 主图必须纯白底，无文字、无水印、无模特半身像（与 Amazon 不同，TEMU 主图更严格）。

## Workflow

### Step 1 — 卖点提炼

从用户输入提取：

- 3 个核心卖点（参数级）
- 3 个情感级卖点（场景 / 生活方式）
- 2 个信任状（资质 / 销量 / 评价）

### Step 2 — 主图 5 张规划

| 顺序 | 类型 | 内容 | 文字 |
|---|---|---|---|
| 第 1 张 | 白底主图 | 产品全貌 85% 占比 | 无文字 |
| 第 2 张 | 卖点图 | 1 个核心功能可视化 | ≤ 8 个词 |
| 第 3 张 | 场景图 | 用户使用场景 | 简短场景描述 |
| 第 4 张 | 尺寸 / 参数图 | 尺寸标注 / 规格表 | 关键参数 |
| 第 5 张 | 包装图 | 包装清单 | 配件列表 |

### Step 3 — 详情页模块排版

按 6 大模块从上到下：

1. 场景图（生活方式情感锚定）
2. 核心卖点 3 连图（参数可视化）
3. 使用步骤 / 安装指引（4-6 步图解）
4. 参数表（双语对照）
5. 资质证书（CE / FCC / FDA 等）
6. 售后保障（退货 / 换货 / 客服响应）

### Step 4 — 退货率导向预期前置

针对高退货品类，在详情页前 3 屏强调：

- 真实尺寸图（与实物对比，避免「偏大偏小」差评）
- 材质近距特写（避免「与想象不符」差评）
- 安装操作动图（避免「不会用」差评）
- 包装清单（避免「配件缺失」差评）

### Step 5 — 多语言适配

每个语种版本需注意：

- 文字长度差异（德语词长，西班牙语长尾）
- 字体兼容性（韩文 / 日文需嵌入字体）
- 排版从左到右（DE / FR / ES）与从上到下（部分语言）
- 文化适配（颜色、模特族裔、节日符号）

### Step 6 — 节庆模板替换

建立素材模板库：

- 主图背景色 → 节庆色（圣诞红绿、复活节粉彩）
- 详情页顶部 banner → 节庆主题
- 视频前 3 秒 → 节庆 hook
- 文案加入节庆关键词（gift / present / holiday）

## Output

| 素材 | 规格 | 数量 | 文件命名建议 |
|---|---|---|---|
| 主图 | 1:1 1000x1000 白底 | 5 | main_01 至 main_05 |
| 视频 | 1:1 1080p 20s | 1 | video_main |
| 详情长图 | 750 宽 | 8-12 张 | detail_01 至 detail_12 |
| 多语言版本 | 同上 | 7 语种 | 追加 _en / _de / _fr 等后缀 |

## Quick Mode

只回答「这个 SKU 详情页怎么排」时，输出 4 行：

1. 主图 5 张内容建议
2. 详情页前 3 屏模块顺序
3. 视频前 3 秒 hook 方向
4. 退货率敏感点提醒（针对该品类）

注意：图片与视频规格会随 TEMU 卖家中心政策调整，上传前用 `web_search` 校准最新规范。
