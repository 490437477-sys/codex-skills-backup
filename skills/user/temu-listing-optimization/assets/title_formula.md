## Quick Sanity Check

For any title you generate, verify:
- [ ] Total length 60-100 chars
- [ ] First 30 chars include primary keyword
- [ ] No keyword repetition
- [ ] No promotional words (best, #1, sale)
- [ ] No emoji or ALL CAPS
- [ ] Natural English (not translated)
- [ ] Age range mentioned if applicable (for compliance)
- [ ] One specific spec (size, count, material) included
