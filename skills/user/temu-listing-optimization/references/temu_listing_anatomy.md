# TEMU Listing Anatomy (Field-by-Field Guide)

> Read this before optimizing any TEMU listing. Each field has different weight, length limits, and best practices.

## Field Map

| Field | Required | Length Limit | SEO Weight | Conversion Weight |
|---|---|---|---|---|
| Main image (1st) | Yes | 1:1, 800x800+ | Low | **Highest** (60% of CTR) |
| Additional images | Yes (min 3, recommend 6-9) | 1:1 or 3:4 | Low | High |
| Video | No (but 3-5x algo boost) | 15-30 sec | Medium | High |
| Title | Yes | **60-100 chars** | **Highest** (60%) | High |
| Key selling points (bullets) | Yes | 5-8, 30-50 chars each | High (20%) | High |
| Rich description | Yes | 500-1000 chars | Medium (15%) | Medium |
| Category attributes | Strongly recommended | 30-50+ fields | **Hidden 30%+** | Low (filter) |
| Price | Yes | n/a | Highest (separate algo factor) | **Highest** |
| Variants (color/size) | If applicable | n/a | High for filter | Medium |
| SKU/barcode | Yes | n/a | Required | n/a |

## Detailed Specifications

### Title
- **Length**: 60-100 characters (TEMU truncates beyond 100)
- **First 30 chars** MUST include primary keyword
- **Structure**: `[Brand] + [Primary keyword] + [Modifier 1] + [Modifier 2] + [Spec] + [Audience]`
- **Do NOT**: Stuff keywords, use promotional words, use ALL CAPS, add emoji
- **Do**: Use natural English, front-load value, mention specific audience

Examples (good):
- `4DOF Robot Arm Kit MG90S Arduino STEM for Teens Ages 13+` (57 chars) ˇŞ TOO SHORT
- `4DOF Robot Arm Kit MG90S Metal Servo Arduino STEM Educational for Teens 13+ DIY` (82 chars) ˇŞ GOOD
- `Robot Arm Kit 4DOF MG90S Servo Arduino-Compatible STEM DIY Educational for Teens Ages 13+` (93 chars) ˇŞ GOOD

### Key Selling Points (Bullets)
- 5-8 bullets, each 30-50 characters
- Format: `[Benefit]: [Feature]` or `[Spec]: [Value]`
- Front-load buyer value, not technical specs
- Use ALL CAPS for first 1-2 words of each bullet for visual scan (TEMU displays them bold)

Examples:
- `STEM LEARNING: 4DOF mechanical arm kit`
- `4 SERVO MOTORS: MG90S metal gear stable`
- `JOYSTICK CONTROL: Plug-and-play USB wired`
- `DIY ASSEMBLY: Improve hands-on skills`
- `GREAT GIFT: For teens ages 13+ educational toy`

### Rich Description
- 500-1000 characters text + 3-5 supporting images
- Structure: Hook (1 sentence) ˇú 3-4 benefit paragraphs (50-100 chars each) ˇú Spec line ˇú Compliance note ˇú CTA
- **Images embedded in description** carry additional conversion weight
- Use 3-5 images inside description: lifestyle, exploded view, in-use, comparison, package contents

Example skeleton:
```
[Product Name] ˇŞ Open the door to engineering and coding for teens ages 13+.

This 4DOF mechanical arm kit features 4 metal-gear MG90S servos for stable, precise movement...

[IMAGE 1: lifestyle - product on desk in use]

Built for hands-on learning: 3D printed structure, Arduino-compatible control board, dual-joystick manual mode plus programmable mode for advanced learners.

[IMAGE 2: exploded view - parts labeled]

Specs: 4DOF (4 degrees of freedom) | 4x MG90S servos | USB wired | 3D printed PLA | Approx. 20cm tall

Package: 1x mechanical arm kit (assembly required), 1x Arduino-compatible control board, 1x joystick shield, 1x USB cable, 1x English manual.

Safety: Recommended for ages 13+. Adult supervision advised. 3D printed parts may have minor cosmetic variations.

[IMAGE 3: package contents]
```

### Category Attributes
- 30-50+ structured fields per category
- Common ones: Brand, Color, Material, Size, Weight, Age Range, Power Source, Country of Origin, Certification, Package Includes
- **Every filled attribute is a hidden ranking signal**
- Skipping attributes is a silent penalty

Most-overlooked high-value attributes:
- Country of Origin (for tariff/buyer trust)
- Age Range (for compliance, child safety)
- Power Source (battery / USB / AC)
- Recommended Uses (multi-select, free traffic)
- Item Package Quantity

### Images
- 6-9 images, 1:1 ratio, 800x800 px minimum
- **First image is everything**: white background, product fills 80% of frame, no text overlay
- 2nd: lifestyle/in-use
- 3rd: scale reference (held in hand, next to coin)
- 4th: feature callout (with graphic arrows or short text)
- 5th: package contents laid out
- 6th: comparison/benefit
- 7-9: variants, warranty, return info

### Video (Optional but 3-5x algo boost)
- 15-30 seconds, MP4, <50MB
- Structure: Hook (0-3s) ˇú Demo (3-20s) ˇú Result (20-25s) ˇú CTA (25-30s)
- Show the product actually working, not just rotating
- Include text overlays for key features

## Common Mistakes

1. **Title over 100 chars** ˇŞ silently truncated
2. **Image with text overlay in first slot** ˇŞ trust penalty
3. **No video** ˇŞ 3-5x fewer impressions vs competitors with video
4. **Bullets that just repeat the title** ˇŞ wasted SEO real estate
5. **Description with no images** ˇŞ half the conversion impact
6. **Missing 50% of attributes** ˇŞ invisible to filter searches
7. **Translated Chinese title** ˇŞ awkward English kills CTR

## When to Update

- **Title**: rarely, only if conversion is poor or new keyword opportunity
- **Bullets**: A/B test quarterly
- **Description**: update seasonally (gift positioning for Q4, etc.)
- **Images**: rotate main image every 14-30 days to fight ad fatigue
- **Price**: adjust weekly based on competitor price tracking
- **Video**: 1-2x per year is enough
