# TEMU Competitor Analysis 馃徑

在 TEMU 平台信息透明度有限的前提下，从同款比价、Listing 拆解、销量信号、Ads 反查、Review 痛点五个维度识别对手弱点与可借鉴点。

## Capabilities

- **同款比价**：抓取 TEMU 同款 Top 5-10 卖家价格、运费、起订量，输出价格带与底价
- **Listing 拆解**：标题结构、主图文案、视频前 3 秒、详情页模块、SKU 矩阵拆解
- **销量信号读取**：识别已拼件数、收藏数、上架时间、活动坑位、品类页排名
- **TEMU Ads 反查**：通过关键词反查竞品投流位置、出价区间、素材风格
- **Review 痛点挖矿**：抓取差评与中评，提炼材质、尺寸、功能、包装四大类高频痛点
- **核价生存力对照**：对比自身供货价与竞品供货价，给出降价空间与替代材料建议

## When to Use

- 用户在 TEMU 发现某个卖得很猛的同款，想知道为什么
- 用户要做某个新品类，需要研究头部商家打法
- 用户被跟卖或被核价压价，需要识别对手供应链优势
- 用户要做季节性活动（黑五 / 圣诞 / 母亲节），需要看去年同期头部素材

## TEMU 竞品分析特有挑战

### 平台透明度有限

- TEMU 没有公开 BSR，统一改用「已拼 X 件」+ 「N 人已购」作为销量代理变量
- 评论数量与星级是公开的，但 Reviewer Profile 与 VP 认证比例需人工抽样
- 销量数字会被平台动态打折显示（展示量 ≠ 真实单量），需用活动节奏交叉验证

### 跟卖与同款化常态

- 同一张主图、不同卖家、价格差 10-30% 是常态
- 判断「谁是源头」需看 SKU 命名规则、包装图、详情页排版一致性
- 价格战常发生在新品上架后 7-14 天，核价机制自动触发

## Workflow

### Step 1 — 锁定竞品池

明确三类对手：直接同款（同 SKU）、替代品（同场景不同款）、品类头部（Top 10 销量）。每类抓 5-10 个 ASIN / 链接。

### Step 2 — 价格与销量快照

用 `web_fetch` 抓取每个竞品的：

- 当前售价、活动价、历史最低价
- 已拼件数、收藏数、上架天数
- 包邮政策、发货模式（Y2 / 半托管 / 海外仓）

### Step 3 — Listing 拆解

按模板拆每个竞品：

- 标题字数、关键词顺序、卖点前置
- 主图前 3 视觉焦点（产品 / 模特 / 包装 / 文字）
- 详情页模块顺序（场景 / 卖点 / 参数 / 售后 / 资质）
- 视频前 3 秒 hook、CTA 出现时机
- SKU 矩阵是否覆盖颜色 / 尺寸 / 套装变体

### Step 4 — Ads 反查

在 TEMU 搜索品类关键词，记录：

- 顶部推荐位（首页顶部 4 个广告坑位）的商家与文案
- 类目页「为你推荐」位的商家轮换规律
- 关键词关联竞品（搜索下拉推荐）

### Step 5 — Review 痛点挖矿

抓竞品 1-3 星评论，按四大维度分类：

- 材质做工：异味、掉色、开线、薄
- 尺寸偏差：偏大、偏小、版型差异
- 功能问题：与描述不符、操作复杂、配件缺失
- 物流包装：挤压、破损、丢件、发货慢

输出痛点频次表，反推自家产品可规避 / 可差异化的卖点。

### Step 6 — 弱点与机会输出

汇总三张表：对手弱点表、可借鉴打法表、差异化定位建议。

## Output

### 同款竞品对照表

| 字段 | 内容 |
|---|---|
| 竞品链接 | TEMU 商品页 URL |
| 卖家名 | 店铺名 |
| 上架天数 | 估算 |
| 当前售价 | USD / EUR |
| 历史最低 | 活动价 |
| 已拼件数 | 销量代理 |
| 发货模式 | Y2 / 半托管 / 海外仓 |
| 评分与评论数 | 4.X / N 条 |
| 主力 SKU 变体 | 颜色 / 尺寸分布 |

### 痛点频次表

| 痛点类别 | 提及次数 | 代表评论 | 改进建议 |
|---|---|---|---|
| 材质 | X 次 | 引文 | 改用 Y 材质 |
| 尺寸 | X 次 | 引文 | 调整尺寸表 |
| 功能 | X 次 | 引文 | 增加 Z 配件 |
| 包装 | X 次 | 引文 | 加固内托 |

### 差异化定位

3 条短句：基于痛点挖矿得出的「我们如何不同」。

## Quick Mode

只回答「这个竞品为什么卖得好」时，输出 5 行：

1. 价格定位（中 / 低 / 极低）
2. 主图与详情最显著的 1 个差异点
3. 销量代理数字（已拼件数 / 评分）
4. 1 个可借鉴动作
5. 1 个弱点 / 我们可超越的点

注意：销量展示数字会被 TEMU 动态调整，单点数据不可作为决策唯一依据。
