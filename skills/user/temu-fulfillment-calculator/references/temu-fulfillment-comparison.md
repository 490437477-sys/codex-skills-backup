# TEMU 三种履约模式深度对比 (Fulfillment Comparison)

> 适用: TEMU 卖家选型决策。覆盖全托管 / 半托管 / 本地对本地三种模式，在库存责任、跨境清关、最后一公里、时效与退货成本上的真实差异。

## 一、三种模式总览

| 维度 | 全托管 Fully Managed | 半托管 Semi-Managed | 本地对本地 Local-to-Local |
|---|---|---|---|
| 库存所在地 | 中国仓 | 海外仓 (FBA / 3PL) | 目标市场本地仓 |
| 头程物流 | 商家集运到 TEMU 中国仓 | 商家自送到海外仓 | 商家自行解决 |
| 跨境清关 | TEMU 负责 | 商家 / 货代负责 | 不涉及 |
| 末端配送 | TEMU 自有 / 合作物流 | TEMU 平台指派 | 商家自选 (TEMU 物流 / 自有) |
| 客服 | TEMU 全权 | TEMU 全权 | 商家承担 |
| 退货 | TEMU 海外仓质检 | 海外仓原路退回 | 商家本地处理 |
| 平台扣点 | 5% - 10% | 8% - 15% | 8% - 12% |
| 资金占用 | 中等 (备货到中国仓) | 高 (备货到海外仓) | 高 (海外仓 + 备货) |
| 适合阶段 | 起步期 / 测款期 | 稳定期 / 爆款期 | 深度运营 / 高客单 |

## 二、全托管 (Fully Managed) 详解

### 2.1 流程
```
商家工厂 → 国内仓 → TEMU 中国集运仓 → 跨境头程 (TEMU 包机 / 海运)
       → TEMU 海外仓 (美西 / 美东 / 欧仓) → 末端配送 (USPS / DHL / 当地物流)
```

### 2.2 责任分工
- 商家承担: 国内仓到 TEMU 中国仓的集运 + 商品质检 / 外箱唛头 / SKU 标签 + 英文 / 多语种 listing 素材 + 售后纠纷举证 (质量问题 / 错发漏发)
- 平台承担: 跨境头程 (海运 / 空运 / 包机) + 海外清关缴税 + 海外仓储调拨 + 末端配送上门 + 客服 + 退货入库二次销售

### 2.3 优势与痛点
- 优势: 资金压力小、运营简单、平台流量倾斜 (新链接默认进入全托管推荐池)、节庆活动优先入选
- 痛点: 核价压力大 (比价算法主战场)、退货率天花板低 (15% 无理由退货)、海外仓销毁成本由商家承担、美国时效 7-15 天

## 三、半托管 (Semi-Managed) 详解

### 3.1 流程
```
商家工厂 → 国内仓 → 跨境头程 (商家自选货代) → 海外仓 (FBA / 3PL / 自建)
       → TEMU 平台派单 → 末端配送 (TEMU 物流网络)
```

### 3.2 责任分工
- 商家承担: 头程物流 + 跨境清关缴税 + 海外仓选址 (Amazon FBA / ShipBob / 自建) + 库存补货预测调拨 + 末端服务商对接 (FedEx / UPS / DHL / 当地邮政) + 海外仓 24h 内出库
- 平台承担: 前端流量 + 搜索权重 + 订单分配 + 末端配送调度 + 客服 + 售后工单 + 退货入库

### 3.3 优势与痛点
- 优势: 美国 3-7 天到货、欧洲 2-5 天；定价比全托管自由；高客单 (>$30) 转化可行；可承接 Amazon FBA 库存溢出
- 痛点: 海外仓备货 ≥ 30 天销量、海运延误 / 清关扣货自担、海外仓原路退回 + 销毁费高、必须配跨境运营团队

## 四、本地对本地 (Local-to-Local) 详解

### 4.1 流程
```
商家工厂 → 跨境头程 → 目标市场本地仓 (美国 / 欧洲 / 日本)
       → 商家自选物流 (USPS / FedEx / 本地 3PL) → 买家
```

### 4.2 责任分工
- 商家承担: 全段头程 + 清关缴税 + 本地仓运营 (WMS + 库存管理) + 末端配送 (自建物流 / 接入 TEMU 物流 API) + 客服 + 退货退款 + 当地税务登记 (美国 EIN / 欧洲 VAT / 日本法人)
- 平台承担: 前端流量 + 平台曝光 + 支付通道 (TEMU Pay / 当地支付) + 平台规则与合规审核

### 4.3 优势与痛点
- 优势: 美国 1-3 天到货 (等同 Amazon Prime)、扣点 8%-12% 毛利率最高、可同时接 Amazon / Shopify / TEMU 多渠道、本地处理退货换货体验好
- 痛点: 进入门槛最高 (本地公司 / VAT / 仓储资质)、资金占用最大 (三段库存)、多市场税务复杂、平台流量权重低于全托管

## 五、五维度对比表

| 维度 | 全托管 | 半托管 | 本地对本地 |
|---|---|---|---|
| 进入门槛 | ★☆☆ | ★★☆ | ★★★ |
| 资金占用 | 中 | 高 | 极高 |
| 时效 (美国市场) | 7-15 天 | 3-7 天 | 1-3 天 |
| 平台扣点 | 5%-10% | 8%-15% | 8%-12% |
| 核价压力 | 极高 | 高 | 中 |
| 退货成本 | 低 | 中 | 高 |

| 团队配置 | 1-2 人 | 3-5 人 | 8-15 人 |

## 六、跨境清关与最后一公里

### 6.1 清关模式

| 模式 | 清关主体 | 缴税方 | HS Code 要求 |
|---|---|---|---|
| 全托管 | TEMU (NVOCC 资质) | TEMU | 商家提供 |
| 半托管 | 商家 / 货代 | 商家 | 商家提供 + 货代审核 |
| 本地对本地 | 商家 | 商家 | 商家提供 + 本地报关行 |

### 6.2 关税成本
- 美国: 800 美元以下免税 (de minimis)，2025 年起逐步取消
- 欧盟: VAT 20%+ + 进口关税 (按 HS Code)
- 英国: VAT 20% + 关税
- 巴西: 进口税 60%+ (高门槛市场)
- 日本: 关税 0%-10% + 消费税 10%

### 6.3 最后一公里服务商

| 市场 | TEMU 自有 | 主要合作 | 平均时效 |
|---|---|---|---|
| 美国 | TEMU Express | USPS / UPS / FedEx / OnTrac | 2-5 天 |
| 欧洲 | TEMU Logistics | DHL / DPD / GLS / Yodel | 1-4 天 |
| 日本 | TEMU 物流 | Yamato / Sagawa / 日本邮政 | 1-2 天 |
| 韩国 | TEMU 物流 | CJ Logistics / 韩进 | 1-2 天 |
| 墨西哥 | Estafeta | FedEx MX / 墨西哥邮政 | 3-7 天 |
| 巴西 | Correios | Total Express / Azul Cargo | 5-10 天 |

## 七、时效与退货成本对比

### 7.1 退货成本结构

| 模式 | 退货入库 | 运费承担 | 销毁成本 | 二次销售比例 |
|---|---|---|---|---|
| 全托管 | TEMU 海外仓 | 平台 | 商家 ($0.5-$2 / 件) | 60%-75% |
| 半托管 | 商家海外仓 | 商家 ($3-$8 / 件) | 商家 ($1-$3 / 件) | 50%-65% |
| 本地对本地 | 商家本地仓 | 商家 ($2-$5 / 件) | 商家 ($0.5-$2 / 件) | 70%-85% |

### 7.2 时效影响
```
转化率 ≈ 基础 CVR × (1 + 0.05 × 每减 1 天到货时效)
```

> 美国市场全托管 (10 天) vs 本地对本地 (2 天)，转化率差异可达 +40%。退货评分: 全托管平台承担但计入店铺退货率，半托管 + 本地对本地均由商家承担并接受平台监督。

## 八、选型决策树

| 阶段 | 单量 | 客单价 | 时效要求 | 推荐 |
|---|---|---|---|---|
| 起步期 0-3 个月 | < 100 单/天/链接 | < $15 | 不敏感 | 全托管 |
| 增长期 3-12 个月 | > 100 单/天 | $15 - $50 | 3-7 天 | 半托管 |
| 成熟期 12 个月+ | > 1,000 单/天 | > $50 | 1-3 天 | 本地对本地 |
| 组合策略 | 测款 + 主推款并行 | 全托管 + 半托管并行 | 节庆爆品选半托管 | 长尾 SKU 用全托管 |

> 本地对本地额外前置条件: 已注册美国公司 / 欧洲 VAT 等本地资质。

## 九、避坑清单

- 全托管不要投高客单 ($30+): 转化率被时效拖累
- 半托管必须签 2 家以上货代: 单家延误会断货
- 本地对本地不要一开始就铺多市场: 先单市场 (美国) 跑通
- 海外仓不要单一渠道: FBA + 3PL 组合降低单点风险
- 不要忽视当地税务: 欧洲 VAT 漏报罚款可达 €10,000+

> 数据综合自公开资料，更新于 2026 年 6 月。具体扣点与时效以 TEMU 官方公告为准，建议用 web_search 比对最新政策。
