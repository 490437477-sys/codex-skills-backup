# Ozon 必填属性参考 (Обязательные атрибуты)

> **重要**: 不同类目的必填属性差异很大。Ozon 后台"Характеристики товара"页面对每个类目有具体要求。
> **上架前请用 `web_search` 验证目标类目当前必填项**,或登录 `seller.ozon.ru` 后台查看实时列表。
> 本文给的是常见类目的高频必填项速查。

## 通用必填项(几乎所有类目)

| 属性 (Атрибут) | 说明 | 示例值 |
|---|---|---|
| Бренд (Brand) | 商品品牌 | XIAOMI, Samsung, Собственное производство |
| Страна производства | 产地国 | Китай, Россия, Турция |
| Артикул производителя | 厂商 SKU | 由卖家填 |
| Штрихкод (EAN-13) | 国际条码 | 6901234567890 |
| Единица измерения | 计量单位 | шт, кг, м, л |
| Вес с упаковкой, кг | 带包装重量 | 0.5 |
| Габариты с упаковкой, см | 带包装尺寸 | 20×15×10 |

## 电子产品(Электроника)

补充必填项:
- Тип подключения (Bluetooth / WiFi / USB / Проводной)
- Совместимость (iOS / Android / Windows)
- Емкость аккумулятора, мАч
- Время работы от аккумулятора
- Цвет (по палитре Ozon)
- Мощность, Вт
- Напряжение, В
- Материал корпуса
- Степень защиты (IPXX) — если применимо

## 服装和配饰(Одежда и аксессуары)

补充必填项:
- Состав материала (хлопок 100%, полиэстер 80% / эластан 20%)
- Размер (по российской / европейской сетке)
- Цвет
- Пол (Мужской / Женский / Унисекс)
- Возрастная группа (Взрослый / Детский)
- Сезон (Лето / Зима / Демисезон)
- Тип ткани
- Модель (название модели)

## 家居用品(Дом и интерьер)

补充必填项:
- Материал
- Цвет
- Размеры (Ш×В×Г, см)
- Вес, кг
- Страна производства
- Назначение (для кухни / для ванной / для спальни)
- Тип установки (настольный / настенный / напольный)
- Особенности (с крышкой / складной / разборный)

## 玩具(Игрушки)

补充必填项:
- Возрастная группа (0+, 3+, 6+, 12+)
- Материал
- Цвет
- Пол ребенка (Мальчик / Девочка / Унисекс)
- Развиваемые навыки (моторика, логика, творчество)
- Тип (конструктор, настольная игра, мягкая игрушка)

> **玩具特别注意**: 必须有 EAC 认证 + 玩具安全技术法规(TR TS 008/2011)。

## 美妆和香水(Красота и парфюмерия)

补充必填项:
- Объем, мл
- Тип кожи (для кремов) / Тип волос (для шампуней)
- Состав (полный INCI список)
- Срок годности, мес
- Страна производства
- Назначение (увлажнение / очищение / anti-age)
- Эффект

> **美妆特别注意**: EAC 强制 + 配方注册(TR TS 009/2011)。

## 食品(Продукты питания)

补充必填项:
- Состав
- Срок годности
- Условия хранения
- Энергетическая ценность (ккал / 100г)
- Белки / Жиры / Углеводы (г / 100г)
- Масса нетто, г
- Страна производства
- Тип упаковки

> **食品特别注意**: 严格监管,需 свидетельство о регистрации。

## 季节性商品

如果商品与特定季节相关(НГ, 23 февраля 等),建议:
- 在描述中标注季节属性(Зимний, Летний, Праздничный)
- 在 Rich Content 加节日使用场景图
- 提前 2 个月调整属性(如 Зимний → Летний 切换)

## 不填的影响

- **少 1 个必填项** → 商品无法上架
- **少 1 个可选推荐项** → 搜索排名降低 5-15%
- **描述/标题含未声明属性** → 可能被判定为虚假陈述,降权

## 验证清单

上架前:
- [ ] 所有必填项已填
- [ ] 颜色、尺寸等枚举值用了 Ozon 官方字典(避免拼写错误)
- [ ] 单位用俄语/国际单位(мл, кг, см, шт)
- [ ] 产地国家准确
- [ ] 条码(EAN-13)有效
- [ ] 包装后重量/尺寸实测过
