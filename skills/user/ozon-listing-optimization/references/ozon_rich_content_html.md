# Ozon Rich Content HTML规范

Ozon "Rich Content" 是商品详情页的 HTML 富文本描述区(类似 Amazon A+ Content)。本参考给出 HTML 结构、样式规则与最佳实践。

## 基础要求

### 允许的 HTML 标签

| 标签 | 用途 | 备注 |
|---|---|---|
| `<h2>`, `<h3>`, `<h4>` | 标题 | H1 不要用(浏览器默认会渲染) |
| `<p>` | 段落 | 主要内容容器 |
| `<ul>`, `<ol>`, `<li>` | 列表 | 突出特性 |
| `<table>`, `<tr>`, `<td>`, `<th>` | 表格 | 技术参数 |
| `<strong>`, `<b>`, `<em>`, `<i>` | 加粗/强调 | 关键字突出 |
| `<br>` | 换行 | |
| `<img>` | 图片 | 必须 https,Ozons3 / 自有CDN |

### 禁止的标签

- `<script>` — 不允许
- `<style>` — 大部分样式被剥离
- `<iframe>` — 仅 Ozon 内部域名允许
- `<form>` / `<input>` / `<button>` — 不允许
- `<a href>` — 外部链接被剥离;如必须保留,需用 Ozon 内部链接结构

### 样式限制

Ozon Rich Content 自动应用样式:
- 默认字体:Systat / Ozon Sans 系
- 默认字号:14-16px
- 颜色:接近黑色(`#1A1A1A` 附近)
- 间距:由 `<p>` / `<ul>` 自动控制

不要写 `style="..."` 大部分会被剥离。

## SEO 优化原则

### 关键字布局

1. **第一段**(H2 之后第一段)必须包含**核心关键字** 1-2 次
2. 每个 H3 小标题必须含一个相关关键字
3. bullet points 的<b>标签内放核心关键词</b>
4. 不要堆砌关键字 — 俄语写作要自然,MTL 中文直接翻译会被识别为低质

### 关键字密度

- 目标:核心关键字出现 3-5 次
- 描述总长度:800-1500 字符(中文/拉丁字符都按字符算)
- 关键字密度:2-4%(自然出现)

### 俄语 SEO 特殊点

1. **Падежи**(格变化)必须覆盖 — 不要只写 "наушники",也要写 "наушников", "наушникам"
2. **同义词** — "беспроводные" + "без провода" + "bluetooth"
3. **长尾词** — 在 FAQ 中回答 "Для чего нужен этот товар?"
4. **本地化** — "Москва", "Санкт-Петербург" 在商品用途中适当提及

## 常见错误

❌ 直接复制 Amazon / Shopify 的英文 HTML
❌ 用机器翻译的俄语(语法错误、生硬)
❌ 只用图片堆砌,无文字内容
❌ 外部链接(被剥离)
❌ 表格无 border / 复杂 CSS
❌ 关键信息全部在图片里(图片 Alt 不算 SEO 内容)

✅ 母语级俄语
✅ 自然的 падежи 变化
✅ FAQ 回答真实买家问题
✅ 表格简明、信息密度高
✅ 关键字布局自然
