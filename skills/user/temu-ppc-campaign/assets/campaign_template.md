## Monthly Decision Matrix

After 30 days, score each product:

| Criterion | Weight | Score (1-10) | Weighted |
|---|---|---|---|
| ROAS vs target | 30% | | |
| ROAS trend (improving/stable/declining) | 20% | | |
| Review velocity (reviews added per week) | 15% | | |
| Inventory turn (units sold / units in stock) | 15% | | |
| CTR vs category benchmark | 10% | | |
| CVR vs category benchmark | 10% | | |
| **Total** | 100% | | **/10** |

**Decision**:
- **8.0+**: Scale ЁЊ increase budget 50%
- **6.0-7.9**: Hold ЁЊ optimize listing, maintain budget
- **4.0-5.9**: Watch ЁЊ fix specific issues, evaluate at 60 days
- **< 4.0**: Kill ЁЊ pause ad, liquidate inventory

## Seasonal Overlay Checklist

### Pre-Q4 (September)
- [ ] Inventory check ЁЊ do we have stock for 3x demand?
- [ ] Budget plan ЁЊ Q4 daily budget = 3x current
- [ ] Listing refresh ЁЊ main image updated for gift positioning
- [ ] ROAS tier pre-test ЁЊ ЮШЖЈдіГЄ for proven winners

### Q4 Peak (November 1 - December 20)
- [ ] Daily monitoring (Black Friday + Cyber Monday week)
- [ ] Budget reallocation to top performers
- [ ] Pause any product with declining CVR (return rate spike)
- [ ] Cut budget for slow-movers (reallocate to winners)

### Post-Q4 (December 20 - January 15)
- [ ] Pause all ads except proven Q4 winners
- [ ] Liquidate slow-mover inventory with limited ads
- [ ] Plan Q1 strategy (lower ad spend, focus on Q1 winners)
- [ ] Year-end review: total ad spend, total ROAS, total profit
