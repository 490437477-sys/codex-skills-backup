# TEMU Review Analyzer

TEMU 评论抓取 + 主题聚类 + 情感分析 + 改进建议，针对 TEMU 特有的「机翻评论」「退货驱动差评」「物流时效抱怨」「核价心理落差」做专项识别。

> **重要前提**：TEMU 评论抓取规则、评价体系、平台对刷评的检测机制可能变化。**所有抓取策略与平台政策在动笔前必须用 web_search 验证最新值**，不要直接套用历史抓取脚本。

## Capabilities

- 按 ASIN / SPU / 关键词 抓取 TEMU 评论（含 1 - 5 星、附图、附视频、买家秀）
- 好评主题聚类：质量 / 价格 / 物流 / 包装 / 售后，识别核心卖点
- 差评主题聚类：破损 / 尺寸不对 / 色差 / 异味 / 功能不符 / 与图片不符
- TEMU 特有模式识别：机翻评论、退货驱动差评、物流时效抱怨、核价心理落差
- 复购信号：NPS 估算、二次购买意愿、品牌词提及
- 产品改进建议：基于高频差评主题给出可执行的改款 / 改包装 / 改详情页建议

## When to Use

- 用户问「TEMU 差评怎么解决」「怎么改产品」
- 用户想做选品调研，想看目标 SKU 的真实买家反馈
- 用户已经上架，差评率偏高，想找到根因
- 用户准备改款 / 升级 SKU，想知道买家最在意什么
- 用户做竞品分析，想看 TOP 竞品的好评与差评结构
- 用户问「TEMU 评分怎么提高」「怎么避免差评」

## TEMU 评论的 5 个独特之处

1. **机翻评论占比高**：很多买家用小语种写评论被机翻成英文，关键词偶有偏差，需要做语言归一化
2. **退货驱动差评**：TEMU 退货政策宽松，部分买家为退货而故意打低分（与产品质量无关）
3. **物流时效抱怨集中**：尾程时效 7 - 15 天远慢于 Amazon，差评里 15% - 25% 是物流相关
4. **核价心理落差**：买家发现同款几天后降价，会回头给差评「降价太快」
5. **图片 / 视频差评有信息量**：TEMU 鼓励买家秀，附图差评通常描述问题更具体

## Workflow

### Step 1 - 锁定目标 SKU
向用户确认：目标 ASIN / SPU / 商品链接 / 关键词搜索，竞品范围（同款 Top 10）。

### Step 2 - 评论抓取
按目标抓取近 90 天评论，目标样本量 ≥ 200 条 / SKU，覆盖 1 - 5 星各档。

### Step 3 - 数据清洗与语言归一化
- 中英机翻文本归一化（去除机翻痕迹）
- 去除模板评论、明显刷评
- 关键词提取 + 词频统计

### Step 4 - 主题聚类
按好评 / 差评分别聚类到 8 - 10 个主题：
- 质量（耐用 / 易坏 / 异味 / 材质）
- 尺寸（偏大 / 偏小 / 与描述不符）
- 色差 / 款式
- 功能（不符合预期 / 比预期好）
- 物流（时效 / 包装 / 破损）
- 价格（便宜 / 值 / 偏贵 / 降价快）
- 售后（响应 / 退换 / 客服）
- 其他

### Step 5 - 情感得分与 NPS
计算净推荐值 NPS（5 星 + 4 星比例 减去 1 星 + 2 星比例）。

### Step 6 - TEMU 特有信号识别
- 退货驱动差评占比（关键词：「return」「refund」「money back」）
- 物流时效抱怨占比
- 核价心理差评占比（关键词：「price drop」「cheaper later」「discount」）
- 机翻评论占比

### Step 7 - 产品改进建议
输出 Top 3 - 5 个改进方向（按差评主题频次 × 严重度排序）：
- 改款建议（尺寸 / 材质 / 颜色）
- 改包装建议（防破损 / 加说明书）
- 改详情页建议（图片 / 视频 / 描述）
- 改售后建议（延长保修 / 增加配件）

## Output

输出主表：`主题 | 好评频次 | 差评频次 | 净情感 | 改进优先级`

附表：
- Top 10 高频关键词（好评 / 差评分开）
- TEMU 特有信号占比（退货 / 物流 / 核价）
- 竞品对比（好评分主题重合度、差评分主题差异）
- 改进建议清单（按 ROI 排序）

## Quick Mode

只输入：1 个 ASIN / SPU 链接。1 个输入 → 输出 Top 5 差评主题 + Top 3 改进建议。

## 常见误区

- **只看平均分不看分布**：TEMU 平均 4.5 但 20% 差评率，比平均 4.2 但 5% 差评率危险得多。
- **忽略附图差评**：附图差评通常描述问题更严重，且买家期望值更高。
- **把物流差评当产品问题**：TEMU 物流时效差是平台问题，卖家很难改，但需要包装防破损补救。
- **不对比竞品评论**：同样的差评主题，竞品处理方式可以直接借鉴。


## 实战 Tips

- **永远跑一次 Quick Mode 再做详细测算**：5 个数字 10 秒出结果，先判断大方向值得做再深入。
- **TEMU 数字每月变**：所有费率 / 扣点 / 核价基线 / 退货率每月都可能微调，建议每 30 天复核一次。
- **多市场分散风险**：单一市场被核价压垮时可以切换到另一市场，履约链路要支持多目的地。
- **保留 10% - 15% 利润缓冲**：所有测算都假设最好情况，实际留出 10% - 15% 的缓冲空间才稳。
