# TEMU Advertising Strategy 馃徑

围绕 TEMU Ads 的搜索广告、推荐位广告、活动坑位三类资源，制定从冷启动到爆品期、从日常到大促的全周期投放策略。

## Capabilities

- **投放目标拆解**：新品冷启动 / 爆品冲量 / 防御拦截 / 节庆抢位 四类目标分策略
- **关键词分层出价**：核心词 / 长尾词 / 竞品词 / 自动词 四层出价模型
- **预算分配**：跨站点、跨活动、跨广告类型的预算配比建议
- **ACOS 目标设定**：按毛利与生命周期阶段设差异化 ACOS 红线
- **大促节奏**：黑五 / 网一 / 圣诞 / 母亲节前中后三阶段打法
- **跨站点组合**：US / DE / FR / UK / JP / KR / MX / BR 多站点投放矩阵

## When to Use

- 用户新品上架，需要冷启动投放预算与关键词
- 用户已有链接但单量不稳，需要诊断投放结构
- 用户准备大促，需要提前 4-6 周的投流节奏
- 用户管理多站点，需要跨站点预算与资源协调

## TEMU Ads 资源地图

### 三类广告位

| 资源位 | 触发场景 | 主要指标 | 适合阶段 |
|---|---|---|---|
| 搜索广告 | 用户搜索品类词 / 品牌词 | CTR / ACOS / 单量 | 全周期 |
| 推荐位广告 | 首页 / 详情页 / 类目页为你推荐 | CTR / CVR / 加购率 | 爆品期 |
| 活动坑位 | 平台秒杀 / 主题活动 / 节日 banner | 坑位产出比 | 大促期 |

### 计费与出价

- 主流为 CPC（按点击付费），部分推荐位支持 CPM
- 出价下限因类目而异（0.05 至 0.30 USD），最高出价受日预算与质量分约束
- 平台对点击率与转化率有动态加权（类似 Amazon 算法）

## Workflow

### Step 1 — 目标与红线确认

向用户确认：

- 投放阶段（冷启动 / 冲量 / 防御 / 大促）
- 单 SKU 目标 ACOS（基于毛利倒推）
- 日预算与总预算上限
- 目标站点与语种

### Step 2 — 关键词分层

按四层搭建：

| 层级 | 类型 | 出价策略 | 数量 |
|---|---|---|---|
| 核心词 | 类目大词（如 wireless earbuds） | 中高 | 5-10 |
| 长尾词 | 场景 / 用途 / 受众组合 | 中 | 20-40 |
| 竞品词 | 同款头部商家品牌词 / 型号词 | 中低 | 5-15 |
| 自动词 | 平台自动匹配 | 由低至高 | 持续跑数据 |

### Step 3 — 预算分配

| 阶段 | 预算结构 | 周期 |
|---|---|---|
| 冷启动（0-14 天） | 70% 核心词 + 20% 自动词 + 10% 长尾 | 积累点击与转化数据 |
| 加速期（15-45 天） | 50% 核心词 + 30% 长尾 + 15% 竞品 + 5% 推荐位 | 提权重、拓词 |
| 爆品期（45 天后） | 40% 推荐位 + 30% 核心词 + 20% 长尾 + 10% 活动 | 抢坑位、防跟卖 |
| 大促前 1 周 | 50% 活动坑位 + 30% 核心词 + 20% 长尾 | 集中预算冲排名 |

### Step 4 — ACOS 节奏管理

按生命周期设差异化红线：

- 冷启动期：ACOS ≤ 毛利 1.2 倍（允许亏损换数据）
- 加速期：ACOS ≤ 毛利 0.8 倍（逼近盈亏平衡）
- 爆品期：ACOS ≤ 毛利 0.5 倍（纯利润贡献）
- 节庆期：临时 ACOS 上浮至 1.5 倍毛利，节后回归

### Step 5 — 大促投放节奏

大促前 4 周 / 2 周 / 当周 / 期间 / 之后：

- 4 周前：测试新素材、扩充关键词词量
- 2 周前：提升核心词出价 20-30%，抢排名
- 当周：日预算提至日常 2-3 倍，监控实时 ACOS
- 期间：每小时看一次数据，及时下线低效词
- 之后：清理活动期临时词，回归正常 ACOS 红线

### Step 6 — 跨站点协调

- 同一 SKU 在多站点同步上架时，预算按站点毛利加权分配
- US 站积累的素材与关键词经验可复用至 UK / CA / AU 英文市场
- DE / FR / ES 等本地化站点需独立测试（受众偏好不同）
- 总预算警戒线：单一站点占比 ≤ 60%，避免单点失败

## Quick Mode

只回答「这个 TEMU 链接该怎么投」时，输出 4 行：

1. 当前阶段判断（冷启动 / 加速 / 爆品 / 节庆）
2. 关键词分层与出价建议
3. 日预算与 ACOS 红线
4. 重点动作 1-2 条

注意：TEMU Ads 出价下限、扣点比例、坑位规则会随平台政策变化，投放前用 `web_search` 校准最新数据。
