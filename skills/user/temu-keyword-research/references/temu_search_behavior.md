# TEMU Search Behavior & Algorithm Cheat Sheet

> **Critical context for keyword strategy.** TEMU is fundamentally different from Amazon. Buyers behave differently, the algorithm rewards different signals, and listing structure is different. Read this before doing keyword research.

## Buyer Behavior Differences

| Behavior | Amazon Buyer | TEMU Buyer |
|---|---|---|
| Search intent | "I know what I want" | "Show me deals in this category" |
| Search query length | 3-5 words | 2-3 words |
| Browsing share | 30% | 70%+ |
| Decision driver | Reviews + brand | Price + first 3 images |
| Brand loyalty | High | Low |
| Repeat purchase | Same brand | Same seller if good experience |

**Implication**: On TEMU, **broad + high-volume + visual** beats **specific + long-tail + text-driven**. Amazon-style long-tail stuffing is less effective.

## TEMU Search Ranking Factors (2026)

Based on seller reports and platform policy reviews, the ranking weight order is:

| Factor | Estimated Weight | Why It Matters |
|---|---|---|
| Price competitiveness | 25% | TEMU''s #1 brand promise is "lowest price" |
| Sales velocity (recent 7/30 days) | 20% | Demonstrates product-market fit |
| Click-through rate (CTR) | 15% | First 3 images + price determine CTR |
| Title keyword match | 12% | Direct matching with buyer query |
| Return/refund rate | 8% | Negative signal, kills ranking fast |
| Rating (4.5+ needed) | 7% | Quality signal |
| Review count | 5% | Social proof |
| Image count + quality | 4% | Main conversion driver |
| Video presence | 3% | Strong conversion + algo boost |
| Seller authority | 1% | Store age, total sales, dispute rate |

**Note**: These weights are estimates from seller communities and may not be exact. TEMU does not publish its algorithm.

## Title vs. Description vs. Backend (TEMU vs. Amazon)

| Field | Amazon | TEMU |
|---|---|---|
| Title | 200 chars, primary SEO | **60-100 chars, HIGHEST weight** |
| Bullets | 5, ~500 chars total | 5-8 short bullets, ~30 chars each |
| Description | 2000 chars, A+ Content | Rich Description (text + images), 500-1000 chars |
| Backend search terms | 250 bytes, no duplicates with title | **DOES NOT EXIST** |
| Subject matter / Attributes | 3-5 fields | **30-50+ structured fields** (color, material, size, etc.) |
| A+ Content | Yes (premium) | No, but Rich Description with images |

**Critical**: All TEMU SEO must go into **Title + Attributes + Category**. There is no hidden field to stuff keywords.

## Title Formula (Tested Best Practices)

```
[Brand (optional)] + [Primary keyword] + [Modifier 1] + [Modifier 2] + [Size/Quantity] + [Audience/Use case]
```

Examples that work on TEMU US:
- `4DOF Robot Arm Kit MG90S Arduino Compatible STEM DIY for Teens`
- `Acacia Wooden Cutting Board with Handle Juice Groove 12x8 inch`
- `Cotton Baby Swaddle Wrap 3-Pack Unisex Soft Muslin 47x47 in`

**Do not**:
- Repeat the primary keyword twice ("Robot Arm Kit Robotic Arm...")
- Use ALL CAPS for non-acronyms
- Include promotional words ("Best", "#1", "Sale", "Free Shipping")
- Add emoji or special characters (reduces trust)

**Do**:
- Use natural English phrasing
- Put the strongest keyword first
- Include audience/use case ("for Teens", "for Kitchen", "for Camping")
- Use specific dimensions and quantities

## Category Attribute Filling (Often Overlooked)

TEMU''s category attributes are structured fields (color, material, size, count, etc.). Filling them completely is a **hidden SEO multiplier** because:
1. It enables filter/facet search (buyers browse by attribute)
2. It signals completeness to the algorithm
3. It improves match rate with long-tail queries

For every listing, fill **all** available attributes. Skipping them is a silent ranking penalty.

## Price Positioning Strategy

| Position | Strategy | Risk |
|---|---|---|
| Lowest in category | Maximum impressions, lowest margin | Race to bottom |
| Median - 10% | Sweet spot for most niches | Sustainable |
| Median + 5-15% | Need strong differentiation (bundle, color, video) | Conversion risk |
| Top quartile | Brand story or premium quality required | Very low CTR |

**Recommended starting position**: Median - 10% for first 30 days, then adjust based on conversion.

## Compliance Keywords (Avoid Triggers)

These words in title/description can trigger CPSC scrutiny or platform downrank:
- "kids" (use "for teens" or "for ages 13+" instead)
- "baby" / "infant" (strict CPSIA)
- "medical" / "therapeutic" (FDA)
- "waterproof" without IP rating (false claim risk)
- "non-toxic" without lab report (FDA / Prop 65)
- Brand names you don''t own (Disney, LEGO, Nike, Apple, etc.)
