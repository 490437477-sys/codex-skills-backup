## 价格带分析

noon 三大市场的 AOV 差异：

| 市场 | AOV (2025-2026) | 主力价格带 |
|------|-----------------|------------|
| KSA | 120-180 SAR | 30-300 SAR |
| UAE | 180-260 AED | 50-500 AED |
| EG | 350-500 EGP | 100-1000 EGP |

最热销价格带：30-200 SAR / 50-300 AED / 100-500 EGP。

低于 30 SAR 客单价太低，平台佣金 + 物流不赚钱。
高于 500 SAR 决策周期长、退货率上升。

## 选品决策清单

新 SKU 上市前确认：

- [ ] 价格带在 30-200 SAR / 50-300 AED / 100-500 EGP
- [ ] SASO / SFDA 认证可达（如需）
- [ ] FBN tier 可发（重量 + 尺寸）
- [ ] 月需求量 ≥ 1000 单（用关键词反推）
- [ ] 类目竞品 ≤ 50 个有效 SKU
- [ ] 有差异化（功能 / 设计 / 价格）
- [ ] 中文 / 英文资料可获取

## 失败案例警示

1. **大件家具**：FBN 仓储成本失控 + 退货运输亏本
2. **品牌仿品**：直接被 noon 关闭账户
3. **化妆品无 SFDA**：全部下架 + 资金冻结
4. **超轻小件**：物流占比过高（占比 > 30% 货值）
5. **季节性 SKU 错过窗口**：错过斋月 / White Friday 备货
