# Ozon Performance 活动类型详解

Ozon Performance(原"Продвижение в поиске"+"Трафареты")有 **4 种主要活动类型**,每种有不同的目标、竞价模型和适用场景。

## 1. Поиск (Search Promotion)

**位置**: Ozon 搜索结果页(商品列表)与搜索结果右侧栏。

**目标**: 拦截搜索特定关键词的买家 — 类似 Amazon Sponsored Products。

**竞价模型**: Максимальная цена клика(Max CPC,按点击付费)。

**匹配类型**:
- **Уточнение (Exact)**: 仅完全匹配关键词(类似 Amazon Exact)
- **Фраза (Phrase)**: 包含关键词的短语(类似 Amazon Phrase)
- **Автоматическая (Auto)**: Ozon 自动匹配相关搜索(类似 Amazon Auto,基于商品+类目)

**最佳使用**:
- 主力活动类型 — 转化率最高
- 用于"купить [товар]"等高商业意图关键词
- 配合 $ozon-keyword-research 输出使用

**典型 CPC 范围**: ₽10-80(各类目中位数)

## 2. Категория (Category Promotion)

**位置**: 类目页面(类目导航 → 子类目 → 商品列表)、类目横幅位。

**目标**: 类目浏览流量 — 类似 Amazon Sponsored Brands 出现在类目页。

**竞价模型**: Max CPC(按点击付费)。

**定向**:
- 选择具体类目(可多选,通常 3-5 个相关类目)
- 可选子类目细分

**最佳使用**:
- 品牌防御 — 占据类目页主要位置
- 新品测款 — 类目页流量大、覆盖广
- 大促期间(Black Friday, 11.11, НГ)— 抢占类目曝光

**典型 CPC 范围**: ₽5-50(类目页点击通常比搜索便宜)

## 3. Трафареты (Banners / Display)

**位置**: 站内多处 — 首页横幅、类目页横幅、商品详情页下方、搜索结果底部。

**目标**: 品牌曝光 + 触达浏览用户 — 类似 Amazon Display / DSP。

**竞价模型**:
- **CPM (按千次展示付费)**: 典型 ₽100-500 / 1000 показов
- **Max CPC**: 部分位置可按点击付费

**定向**:
- 类目定向
- 关键词定向(类似搜索)
- 受众定向(Ozon 用户行为)

**最佳使用**:
- 大促前品牌曝光
- 高利润商品的"漏斗上层"
- 已稳定的爆款 — 维持品牌可见度

**典型 CPM 范围**: ₽100-500(因位置而异)

## 4. Карточка товара (Product Page Promotion)

**位置**: 出现在**竞品**商品详情页 — 类似 Amazon Product Targeting。

**目标**: 拦截已经在看竞品的买家 — 高意向拦截。

**竞价模型**: Max CPC(按点击付费)。

**定向方式**:
- **По конкретным товарам**: 指定 1-50 个具体竞品 SKU
- **По категориям**: 整个类目所有商品详情页
- **По брендам**: 竞品品牌的全店商品页

**最佳使用**:
- 竞品为非品牌大卖(拦截他们的高转化流量)
- 类目集中度高、Top 10 占据 70% 销量
- 商品差异化明显(性能 / 价格 / 包装)

**典型 CPC 范围**: ₽5-30(竞品页面点击率较低,但转化率高)

## 活动组合策略

### 新品发布(Launch)

| 活动 | 优先级 | 预算占比 | 目标 |
|---|---|---|---|
| Поиск Exact | P1 | 40% | 拦截高意图搜索 |
| Поиск Phrase/Auto | P2 | 25% | 长尾发现 |
| Категория | P3 | 20% | 类目曝光 |
| Карточка товара | P4 | 15% | 拦截竞品 |

### 成长期(Growth)

| 活动 | 优先级 | 预算占比 | 目标 |
|---|---|---|---|
| Поиск Exact | P1 | 50% | 扩大已验证关键词 |
| Поиск Phrase/Auto | P2 | 15% | 新长尾测试 |
| Категория | P3 | 15% | 维持类目位置 |
| Карточка товара | P4 | 20% | 强力拦截 |

### 成熟期(Mature)

| 活动 | 优先级 | 预算占比 | 目标 |
|---|---|---|---|
| Поиск Exact | P1 | 60% | 防守核心关键词 |
| Поиск Phrase/Auto | P2 | 5% | 关闭或最小化 |
| Категория | P3 | 15% | 品牌防御 |
| Карточка товара | P4 | 20% | 持续拦截 |

### 防御期(Defense)

| 活动 | 优先级 | 预算占比 | 目标 |
|---|---|---|---|
| Поиск Exact | P1 | 70% | 死守核心词 |
| Категория | P3 | 10% | 维持类目 |
| Карточка товара | P4 | 20% | 反击竞品 |

## 活动目标 vs 营销漏斗

```
认知 (Awareness)        → Трафареты + Категория
考虑 (Consideration)    → Поиск Phrase + Карточка товара
决策 (Decision)         → Поиск Exact
复购 (Retention)        → (通常靠品牌 / Ozon Карта,不是广告)
```

## 关键差异对比

| 维度 | Поиск | Категория | Трафареты | Карточка товара |
|---|---|---|---|---|
| CPC 范围(₽) | 10-80 | 5-50 | CPM 100-500 | 5-30 |
| 转化率 | 中-高 | 中 | 低 | 中-高 |
| 流量规模 | 中 | 高 | 极高 | 低 |
| 适用阶段 | 全周期 | 全周期 | 大促 | 成长期+ |
| 关键词定向 | ✅ | ❌ | ✅ | ❌(竞品) |
| 类目定向 | ✅ | ✅ | ✅ | ✅ |
| 受众定向 | ❌ | ❌ | ✅ | ❌ |
| НДС 代扣 | ✅ | ✅ | ✅ | ✅ |

## 实战建议

1. **不要在 Launch 期用 Трафареты** — CPM 太贵,转化漏斗太长
2. **Карточка товара 需要先有 listing 质量** — 竞品页流量来了但 listing 不行,等于烧钱
3. **Поиск Auto 应该有限度** — 自动匹配容易触发无关搜索,快速消耗预算
4. **大类目(如电子) 优先 Поиск Exact** — 长尾词多,精确匹配更高效
5. **小类目(如服饰) Категория + Карточка 性价比更高** — 长尾搜索量本身有限

> **重要**: Ozon Performance 内部规则、竞价模型、佣金率每年都可能调整。所有费率使用前用 `web_search` 验证。
