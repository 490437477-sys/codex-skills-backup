# noon 销量估算基准（无公开 BSR，用评论速度反推）

noon 没有公开 BSR。但评论速度、SKU 数量、上架时长是更可靠的销量代理指标。

## 评论-销量比（MENA 基准）

| 品类 | 评价/销量比 | 备注 |
|------|-------------|------|
| 服饰鞋帽 | 1:5-10 | 评论率高（买家爱晒图） |
| 美妆个护 | 1:10-15 | 评论率中 |
| 母婴儿童 | 1:15-25 | 评论率较低 |
| 电子产品 | 1:15-25 | 评论率低（除非爆款） |
| 家居厨房 | 1:15-20 | 评论率中等 |
| 运动户外 | 1:10-20 | 中等 |
| 食品杂货 | 1:20-50 | 评论率最低 |

反推公式：
```
月销量 ≈ (总评论数 / 上架月数) × 反向比例
```

例：某 SKU 12 个月累计 300 评论，月销量 ≈ 300/12 × 12 ≈ 300 件/月（服饰 1:12 比例）。

## 关键词需求估算

Google Trends + noon 自动补全 + Google Keyword Planner 估算需求池：

```
月度需求池 = 月搜索量 × CTR × CVR × noon 市占
```

| 指标 | 数值范围 |
|------|----------|
| CTR (搜索 → 点击 SKU) | 5-15% |
| CVR (点击 → 购买) | 1-5% |
| noon 市占（占 MENA 总电商） | 30-50% |

例：月搜 10,000 × 10% CTR × 3% CVR × 40% noon 市占 = 12 单/月 per SKU

## 反向推算器

知道 SKU 的：

- 月销量估算 = 评论数 / 上架月数 / 评价率
- 评论月增 = 同上
- 月营收 = 月销量 × AOV × (1 - COD 跳单率)

## 快速估算公式

不需要评论数据时的粗估：

```
月销量 ≈ 类目月需求量 × 类目集中度 × noon 市占 × 卖家份额
```

例：
- 类目月需求：50,000 单
- 类目集中度（前 10 名占）：40%
- noon 市占：35%
- 你在前 10 内的份额：5%

= 50,000 × 40% × 35% × 5% = 350 单/月

## 高销量 SKU 的常见特征

1. **价格带合适**：SAR 30-200 是 noon 主力价格带
2. **FBN**：享受快速承诺
3. **评论 > 100**：社交证明
4. **Sponsored**：站内广告持续投入
5. **图片专业**：白底主图 + 场景图 + 视频
6. **Rich Content**：5 模块以上
7. **A+ 阿拉伯语 Listing**：双语支持
